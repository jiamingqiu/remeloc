Then we construct response by adding 10% of noise to the squared distance. In the data frame, one row for one edge, where the last two columns are index of nodes correspond to this edge.

Next, “fit” a model, namely specify local regression structures. Here we specify that response “y” should be tied to nodes indexed by columns “p1” and “p2”, coordinates of those nodes are supplied as well. Further, we require the algorithm to use no more than \(10^4\) edges locally.

fit <- fitMetric(
  y ~ p1 : p2, 
  data = dat, coord = obsv.coord, 
  optns = list(n.local = 10^4)
)

Then we estimate metric tensor at targeted points using the model. This could take a while. Estimation here is done on a \(128 \times 128\) grid.

target <- manifold$genGrid(128) * coef.shrink
# parallel estimation
system.time({
  N.CORES <- 20
  doParallel::registerDoParallel(cores = N.CORES)
  st.time <- proc.time()
  esti.metric <- foreach(
    w.target = split.data.frame(
      target, # %>% head(2 * N.CORES), 
      cut(seq(nrow(target)), N.CORES)
    ),
    .export = 'fit', .packages = 'remeloc', .errorhandling = 'pass'
  ) %dopar% {
    estiMetric(w.target, fit)
  }
  esti.metric <- unlist(esti.metric, recursive = F)
})

Next, consider the geodesic curve. However, the current estimate is locally constant with zero derivatives, which means one cannot plug the “estiMetric” directly into “getGeodesic” function, additional interpolation step is needed. Currently the “approxfunMetric” uses local polynomial to provide a smoothed estimated tensor. However, it is not interpolation, be careful when using it.

approx.metric <- approxfunMetric(target, esti.metric)
geo.f.true <- getGeodesic(manifold$metric, d = d)
geo.f.esti <- getGeodesic(function(x){estiMetric(x, fit)[[1]]}, d = d)
geo.f.approx <- getGeodesic(approx.metric, d = d)
df.geo <- bind_rows(
  geo.f.true(rep(0.75, d), c(-1, 0)) %>% as.data.frame %>% as_tibble %>%
    mutate(what = 'true'),
  geo.f.esti(rep(0.75, d), c(-1, 0)) %>% as.data.frame %>% as_tibble %>% 
    mutate(what = 'estimated'),
  geo.f.approx(rep(0.75, d), c(-1, 0)) %>% as.data.frame %>% as_tibble %>% 
    mutate(what = 'approx')
)
df.geo %>% ggplot() +
  geom_path(aes(x = x1, y = x2, color = what)) +
  coord_fixed()
